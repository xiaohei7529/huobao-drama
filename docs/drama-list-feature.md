## 🚀 使用方法

### 访问页面

1. 启动开发服务器
   ```bash
   npm run dev
   ```

2. 访问短剧列表
   - 方式 1: 导航栏 → 短剧列表
   - 方式 2: 首页 → 热门推荐 → 查看更多
   - 方式 3: 直接访问 `#/drama-list`

### 功能演示

1. **切换视图**: 点击右上角网格/列表按钮
2. **筛选短剧**: 选择分类、状态、排序
3. **搜索**: 输入剧名回车搜索
4. **分页**: 选择每页数量或翻页
5. **操作**: 点击卡片查看详情，或点击播放/收藏按钮
